# Grind Landing Page

A minimal, editorial-style landing page for a boutique coffee shop. 

## Features
- **Clean Design**: Built with standard HTML5 and Tailwind CSS (via CDN).
- **Dark Mode**: Fully implemented dark mode toggle that persists in local storage and respects system preferences.
- **Responsive**: Adapts perfectly to mobile and desktop screens, with a collapsing mobile menu.
- **Typography**: Uses 'Instrument Serif' for headers and 'Inter' for body text.

## How to run
Since this uses Tailwind via CDN, there is no build step. Simply open `index.html` in your browser or use a local server like Live Server in VS Code!
